# suse-ai-aws

This repository contains OpenTofu and Terraform modules to deploy **SUSE AI** infrastructure on AWS.

## Projects

- `basic-ec2-sles`: A basic EC2 instance deployment using SUSE Linux Enterprise Server (SLES). Compatible with Terraform and OpenTofu.
- `cluster-ec2-sles`: A deployment of two SLES instances in a Cluster Placement Group, optimized for low-latency networking and GPU-based workloads. Compatible with Terraform and OpenTofu.
