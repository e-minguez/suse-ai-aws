# Terraform AWS AI

This repository contains Terraform modules and examples for deploying AI-related infrastructure on AWS.

## Projects

- `basic-ec2-sles`: A basic EC2 instance deployment using SUSE Linux Enterprise Server (SLES).
- `cluster-ec2-sles`: A deployment of two SLES instances in a Cluster Placement Group, optimized for low-latency networking and GPU-based workloads.
